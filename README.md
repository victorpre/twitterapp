# Client Using Twitter API

Ruby version: 2.4.0

Rails version: 5.0.1

Available in:
https://twitter-api-example.herokuapp.com/

For development:

- Clone the repository and add your API keys in the `twitterapp/config/secrets.yml` file.
- `bundle install`
- `rake db:{create,migrate}`

For tests:

- `bundle exec rspec`
